# VibeTest

Конструктор и прохождение тестов в браузере. В **guest**-режиме приложение работает автономно (localStorage, импорт/экспорт JSON); в **full**-режиме добавляются регистрация, облачное хранение тестов, публикация и история прохождений на ASP.NET Core + SQLite.

Стек: React + TypeScript + Vite (`vibetest.client`), ASP.NET Core 10 (`VibeTest.Server`).
